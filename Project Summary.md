# Project Summary

This repository presents the Giverly UAE Expansion capstone project, focused on designing a fintech-for-good solution that integrates cashback rewards, charitable donations, and user engagement features for the UAE market.

My role in this project included:
- Sprint planning and Agile coordination
- Persona, empathy map, and customer journey development
- Requirements prioritisation and MVP thinking
- Prototype contribution and iterative refinement
- Reporting and presentation support
